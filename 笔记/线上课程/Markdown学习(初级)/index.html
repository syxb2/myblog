    Markdown学习笔记(初级)

    （不要用预览模式浏览）

    1. 写markdown的第零步

    我建议大家写 Markdown 的时候都用半角标点，即英文标点,如〝，.7!0口：；〝•旦每个半角标点在文本使用时加上后置空格,符合英文标点的书写规范，也更加美观.

    1. 标题

    一级标题

    二级标题

    三级标题

    等等

    1. 强调[用**或__包围] 或者选中后按下⌃B

    欢迎报考电子科技大学

    1. 斜体[用*或_包围] 或者选中后按下⌃I

    欢迎报考电子科技大学

    1. 删除线[用~~包围]

    ~欢迎报考沧州市第一中学~

    1. 代码[用`（键盘左上角的点）包围]

    hello world

    1. 代码块[按三个`并敲回车]

    代码块里可以选择语言，其会根据语言来自动高亮各个语句．在选择语言后```会变为十对应语言（如c）。

    要想显示行数的话，一般要在Typora 的设置里勾上这个显示行数的选项

    int main() {
return 0;
}
    1. 引用[> + 空格 前置] 引用是可以嵌套的

    欢迎报考

    电子科技大学

    1. 无序列表[“-”或”+” + 空格 前置] 三种前置符都可以，敲回车会自动补全，可在Typora 设置中调整补全的符号，敲回车后按下 Tab 会缩进一级．

    2. 我叫syxb

    3. 有序列表[数字 + . + 空格 前置] 敲回车会自动补全，敲回车后按下 Tab 会缩进一级．

      1. 我是
      2. syxb

    4. 链接[常用[] + () 分别包别文本与链接] (注意：文内跳转为扩展用法）

    [来看看我贫瘠的仓库罢] (https://github.com/sakiyary)

    1. 任务列表[“- [ ]” + 空格 前置] 用x代替[]中的空格来勾选任务列表．在Typora中可以直接用鼠标左键单击勾选框，

      • [] 刷b站

    2. 表格[用”|”绘制表格边框] 在第一行用|分开写上表头 在第二行需要打上你的对齐方式以及文本长度,用引号来代表对齐方式 第三行往后就全是你所要打的数据

    |学号|姓名|年龄| |:---|:---:|---:| (引号的位置代表着 左对齐，居中，右对齐） |1234512345|张三|24|

    1. 图片[直接拖进来或者复制粘贴]

    ! [name] (图片的位置)（类似链接的写法）

    1. 分割线[按三个*或-或_并敲回车] 由于 * 与_均会自动补全，所以我觉得-最为方便

    —-

    1. Emoji表情

    对于其余普通的 Markdown 文本编辑器，可以直接将 Emoji 表情复制进来，这是直接硬编码的（刻进DNA里）
