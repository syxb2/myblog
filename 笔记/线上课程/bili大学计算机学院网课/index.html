    二进制语 汇编语言

    1. 创建源文件

    头文件: xxxx.h

    int main()
{
    return 0;

    第三节. 初识数据类型

    1. 数据类型

    //计算机语言 - 写程序 - 解决生活中的问题

    //必须有能力描述生活中的问题

    float - 单精度浮点数 - 4 byte
double - 双精度浮点数 - 8 byte
    //小数 - 浮点数（因为小数点可以左右浮动15.6=1.56*10^1）
int - 整型 - 4 byte
short - 短整形 - 2 byte
long - 长整型 - 4 byte
    //规定sizeof(long) >= sizeof(int) 即可
long long - 更长的整型 - 8 byte
char - 字符数据类型 - 1 byte

//每种数据类型所占用空间大小固定，其所占空间大小决定了它储存能力的上限
//故（对于同种类型（如整型））要合理选用不同数据类型以使空间利用率最高

    ### 2. 每种类型的大小（指所占空间）是多少？

    #### 打印一个整数与打印大小

    ``` int main() { printf("%d\n", 100); //打印整数100 //%d - 指定格式：打印一个整数 //%lf - 打印一个浮点数

    int age = 10；
printf("%d\n", age)；
//打印age的值

printf(%d\n, sizeof(char));
//打印char所储存的字符的大小
//sizeof - 是一个关键字 - 也是c语言中的一个操作符 - 作用是计算类型或者变量所占空间的大小

    }

    
sizeof的单位是：字节（byte）

    注：
    计算机中常见的单位：（由大到小）

    bit - 比特位 - 最小
        //计算机中识别二进制 
        //一个bit存放一个二进制位

    byte - 字节
        //1 byte = 8 bit

    1 kb = 1024 byte
    1 mb = 1024 kb
    1 gb = 1024 mb
    1 tb = 1024 gb
    1 pb = 1024 tb

#### 为什么单是整型就分为int、short、long、long long呢？

每种数据类型所占用空间大小固定，其所占空间大小决定了它储存能力的上限

故（对于同种类型（如整型））要合理选用不同数据类型以使空间利用率最高
