    Web前端

    前言

    一定要动手实践

    什么是网页

    网站是多个网页的集合

    网页是网站中的一页，通常是HTML格式的文件，它通过浏览器来阅读

    通常网页的格式是.html或.htm

    网页是怎么来的？

    网页由网页元素组成，这些元素是利用html标签描述出来的，然后通过浏览器解析给用户

    常用的浏览器

    浏览器是网页显示，运行的平台

    如：IE，火狐，chrome，Safari，opera等

    浏览器内核（渲染引擎）

    常用的浏览器内核：

    浏览器内核负责读取网页内容，整理讯息，计算网页的显示方式并显示页面。

    Web标准

    Web标准是由W3C组织和其他标准化组织制定的一系列标准的集合，W3C（万维网联盟）是国际最著名的标准化组织。

    为什么需要Web标准？

    为了让不同浏览器面对一个html文件时显示出的内容相同

    Web标准的构成

    包括结构，表现，行为三个方面

    html属于结构

    CSS属于表现，表现可以让页面看起来更漂亮

    javascript（即js）属于行为，可以让网页有交互效果

    结构会更重要一些

    Web标准提出的最佳体验方案：结构，样式，行为相分离

    简单理解：结构写到HTML文件中，表现写到CSS文件中，行为写到JavaScript文件中

    HTML

    什么是html

    HTMl指的是超文本标记语言，用来描述网页的

    它是一种标记语言，标记语言是一套标记标签

    html页面也叫html文档

    超文本

    • 可以加入图片，声音，动画等内容（超越文本的限制）
    • 它还可以从一个文件跳转到另一个文件，与世界各地主机的文件连接（超级链接文本）

    html语法规范

    基本语法概述

    • 所有的标签都需要写在<>中。
    • 标签大部分都是成对出现的，如<html>和</html>，其中<html>叫做开始标签，</html>叫做结束标签，整体叫做双标签。
    • 也有单标签，如<br />，注意其中会有一个空格。

    标签关系

    1. 包含关系
    <head>
    <title> </title>
</head>

    也叫父子关系（head是title的爸爸，title是head的儿子）

    1. 并列关系
    <head> </head>
<body> </body>

    也叫兄弟关系

    标签属性

    标签的属性必须用""包围

    html基本结构标签

    第一个html网页

    每个网页都会有一个基本的结构标签（也称为骨架标签），页面内容也是在这些基本标签上书写。

    <html>
    <head>
        <title>我的第一个页面</title>
    </head>
    <body>
        我是body
    </body>
<html>

    其中：

    • html是页面中最大的标签，也叫根标签
    • head是页面中的头部标签
    • title是网页的标题，也叫做文档标题，显示在浏览器最上方的的标签页中
    • body是网页的主体，包含了文档的主要内容，网页的主要内容就是放到body标签里
    • body和head是并列关系

    解释vscode工具生成骨架中的新增代码

    <!DOCTYPE>标签（doc type）

    是文档类型声明标签，作用就是告诉浏览器使用哪种html版本来显示网页

    <!DOCTYPE html>的意思就是当前页面采取的是HTML5版本来显示网页

    • 此标签必须写到html文件的最前面
    • 此标签不属于html标签，它叫做文档声明标签
    lang语言种类

    用来定义当前文档显示的语言

    <html lang = "en" >
    • en定义语言为英语
    • zh-CN定义语言为中文
    • 其主要作用是提醒，通知作用
    字符集（字符编码格式）

    字符集（Characterset）是多个字符的集合。以便计算机能够识别和存储各种文字。

    在<head>标签内，可以通过<meta>标签的charset属性来规定HTML文档应该使用哪种字符编码。

    <meta charset="UTF-8" />

    charset常用的值有：GB2312、BIG5、GBK和UTF-8，其中UTF-8也被称为万国码，基本包含了全世界所有国家需要用到的字符.

    html常用标签

    标签语义：

    简单来说就是标签的含义，即这个标签是用来干什么的

    根据标签的语义，在合适的地方给一个最为合理的标签，可以让页面结构更加清晰

    标签种类：

    1. 标题标签

    为了使网页更具有语义化，我们经常会在页面中用到标题 标签。HTML提供了6个等级的网页标题，即<h1> - <h6>。

    语义：作为标题使用，并且依据重要性递减

    如：

    <body>
    <h1>标题一共六级选，</h1>
    <h2>文字加粗一行显。</h2> 
    <h3>由大到小依次减，</h3>
    <h4>从重到轻随之变。</h4>
    <h5>语法规范书写后，</h5>
    <h6>具体效果刷新见。</h6>
</body>

    其中，h是head的缩写

    特点：

    • 加了标题的文字会变的加粗，字号也会依次变大。
    • 一个标题独占一行。

    2. 段落标签

    在网页中，要把文字有条理地显示出来，就需要将这些文字分段显示。在HTML标签中，<P>标签用于定义段落，它可以将整个网页分为若干个段落。

    <body>
    <p>
        这是第一段文字
    </p>
    <p>
        这是第二段文字
    </p>
</body>

    特点：

    • 文本在一个段落中会根据浏览器窗口的大小自动换行。
    • 段落和段落之间保有空隙

    其中：p是paragraph的缩写

    3. 换行标签

    在HTML中，一个段落中的文字会从左到右依次排列，直到浏览器窗口的右端，然后才自动换行。如果希望某段文本强制换行显示，就需要使用换行标签＜br />。

    如：

    <body>
    <p>
        这是第一段文字，这是第一行 <br /> 这是第二行
    </p>
</body>

    其中：br为单词break的缩写

    标签语义：强制换行

    特点：

    • 换行标签是一个单标签
    • <br />标签只是简单地开始新的一行，跟段落不一样，换行后行与行之间没有空隙。（换行不是分段）

    4. 文本格式化标签

    在网页中，有时需要为文字设置粗体、科体或下划线等效果，这时就需要用到HTML中的文本格式化标签，使文字以特殊的方式显示。

    标签语义：突出重要性，比普通文字更重要

    • 加粗：<strong></strong>或<b></b> - 更推荐使用<strong>标签，语义更强烈
    • 倾斜：<em></em>或<i></i> - 更推荐使用<em>标签，语义更强烈
    • 删除线：<del></del>或者<s></s> - 更推荐使用<del>标签，语义更强烈
    • 下划线：<ins></ins>或<u></u> - 更推荐使用<ins>，语义你更强烈

    5.
    和标签

    <div>和<span>是没有语义的，它们就是一个盒子，用来装内容的，用来布局页面

    如：

    这就是一个盒子

    <div>这是头部</div>
<span>今日价格</span>

    其中：

    div是division的缩写，表示分割、分区。span意为跨度、跨距。

    特点：

    • <div>标签用来布局，但是一行只能放一个<div>，即<div>标签中的内容单独占一行。大盒子
    • <span>标签用来布局，一行上可以有多个<span>。小盒子

    6. 图像标签和路径

    图像标签

    在HTML标签中，<img>标签用于定义HTML页面中的图像。

    格式：<img src="图像URL" />

    如：

    <body>
    图像标签的使用：<br />
    <img src= "img.jpg">
    <h4>
        alt 的使用
    </h4>
    <img src= "img.jpg" alt= "我是pink老师">
</body>

    其中：

    • img为单词image的缩写，意为图像
    • src是<img>标签的必须属性，它用于指定图像文件的路径和文件名。
    • 图像标签是单标签
    图像标签的属性

    所谓属性，简单理解就是属于这个图像标签的特性。

    • src - 指定图片的路径和名称
    • alt - 替换文本 - 图像显示不出来的时候用文字替换
    • title - 提示文本 - 鼠标放在图像上，显示的文字
    • width - 设定图像宽度
    • height - 设定图像的高度
    • border - 设定图像边框宽度

    注意：

    • 图像标签可以拥有多个属性，必须写在标签名的后面
    • 属性之间不分先后顺序，标签名与属性、属性与属性之间均以空格分开
    • 属性采取键值对的格式，即key= "value"的格式，属性= "属性值"
    • 实际开发当中，一般宽度和高度只设置一个即可，另一个会自动等比例缩放
    • border（边框）一般在css中设定，而不是从img的属性中设定

    图像路径
    目录文件夹和根目录：

    实际工作中，我们的文件不能随便乱放，否则用起来很难快速的找到他们，因此我们需要一个文件夹来管理他们。

    目录文件夹：存放页面所需要的相关素材。如：html源文件、图片（图片常常会放在目录文件夹下的一个新的文件夹里）等。

    根目录：打开目录文件夹的第一层就是根目录

    路径
    1. 相对路径

    即，以引用文件所在位置为参考基础，而建立出的目录路径。简单来说就是图片相对于HTML页面的位置

    • 同一级路径：直接写图像的文件名即可
    • 下一级路径：如：images/img.jpg。其中，images文件夹在根目录下（与html文件处于同一目录下）。

    • 上一级路径：如：../img.jpg。其中，img.jpg在根目录的上一级。再如：../../img.jpg。其中，img.jpg在根目录的上上级。

    • 绝对路径

    是指目录下的绝对位置，直接到达目标位置，通常是从盘符开始的路径（mac上一般从Uesrs开始）。

    例如，"D:\web\img\logo.gif"或完整的网络地址"http//www.itcast.cn/images/ogo.gif"

    注意：相对路径中的斜杠是/，而绝对路径中的斜杠是\。

    7. 超链接标签

    在HTML标签中，<a></a>标签用于定义超链接，作用是从一个页面链接到另一个页面，即点击一个链接后会跳转到指定的页面。

    语法规范
    <a href="跳转目标" target="目标窗口的弹出方式"> 文本或图像 </a>

    其中：

    • a是单词anchor的缩写，意为：锚
    • <a>的两个属性：
    • href：用于指定链接目标的url地址（必须属性）
    • target：用于指定链接页面的打开方式，其中_self为在当前窗口打开（默认值），_blank为在新窗口中打开
    链接分类
    1. 外部链接。例如：<a href="http://www.baidu.com">百度</a>
    2. 内部链接：网站内部页面之间的相互链接，直接链接内部页面名称即可。例如：<a href="index.html">首页</a>
    3. 空链接：如果当时没有确定链接目标时，就使用空链接。如：<a href="#">首页</a>
    4. 下载链接：如果href里的地址是一个文件或者压缩包，会下载这个文件。即：<a href="文件url">下载文件</a>
    5. 网页元素链接：网页中的各种网页元素，如文本、图像、表格、音频、视频等都可以添加超链接。如：<a href="http://www.baidu.com"><img src="img jpg"/></a>
    6. 锚点链接：
      • 找到目标位置标签，里面添加一个id属性 = 名字。如：<h3 id="two">第二集介绍</a>
      • 在链接文本href属性中，设置属性值为#名字的形式，如<a href="#two">第二集</a>

    列表

    表格是用来显示数据的，那么列表就是用来布局的。

    列表最大的特点就是整齐、整吉、有序，它作为布局会更加自由和方便。

    根据使用情景不同，列表可以分为三大类：无序列表、有序列表和自定义列表。

    无序列表

    <ul>标签表示HTML页面中项目的无序列表，一般会以项目符号呈现列表项，而列表项使用<li＞标签定义。

    如：

    <ul>
    <li>列表项1</li>
    <li>列表项2</li>
    <li>列表项3</li>
    ...
</ul>

    其中：

    • 无序列表的各个列表项之间没有顺序级别之分，是并列的。
    • <ul></ul>中只能嵌套<li></li>，直接在<ul></ul＞标签中输入其他标签或者输入文字的做法是不被允许的。
    • <li></li>之间相当于一个容器，可以容纳任何元素。
    • 无序列表会带有自己的样式属性，但在实际使用时，我们会使用CSS来设置。

    有序列表

    有序列表即力有排列顺序的列表，其各个列表项会按照一定的顺序排列定义。

    在HTML标签中，<ol>标签用于定义有序列表，列表排序以数字来显示，并且使用<li>标签来定义列表项。

    <ol>
    <li>列表项1</li>
    <li>列表项2</li>
    <li>列表项3</li>
    ...
</ol>

    其中：

    • <ol></ol>中只能嵌套<li></li>，直接在<ol></ol＞标签中输入其他标签或者输入文字的做法是不被允许的。
    • <li></li>之间相当于一个容器，可以容纳任何元素。
    • 有序列表会带有自己的样式属性，但在实际使用时，我们会使用CSS来设置。

    不常用

        </dd>

    CSS

    简介

    CSS的主要使用场景就是美化网页，布局页面的

    html的局限性

    说起HTML，这其实是个非常单纯的家伙，他只关注内容的语义。比如<h1>表明这是一个大标题，<P>表明这是一个段落，<img>表明这儿有一个图片，<a>表示此处有链接。

    CSS简介

    CSS是层叠样式表（CascadingStyleSheets）的简称。

    有时我们也会称之为CSS样式表或级联样式表

    CSS也是一种标记语言

    CSS主要用于设置HTML页面中的文本内容（字体、大小、对济方式等）、图片的外形（宽高、边框样式、边距等）以及版面的布局和外观显示样式。

    CSS最大价值：由HTML专注去做结构呈现，样式交给CSS，即结构（HTML）与样式（CSS）相分离。

    CSS语法规范（体验）

    CSS规则由两个主要的部分呢构成：选择器以及一条或多条声明。

    如：

    <head>
    <style>
        /* 选择器 {样式} */
        /* 给谁改样式 {改什么样式} */
        p {
            color: red;
            /* 修改了文字大小为12像素 */
            font-size: 12px;
            /* px为像素的意思，font为字体的意思 */
        }
    </style>
</head>

<body>
    <p>
        标题
    </p>
</body>
    • 选择器是用于指定CSS样式的HTML标签，花括号内是对该对象设置的具体样式
    • 属性和属性值以键值对的形式出现（即：属性: 属性值;）

    选择器

    作用

    选择器（选择符）的作用：即选择要修改的对象标签。

    分类

    选择器分为基础选择器和复合选择器两个大类。

    • 基础选择器是由单个选择器组成的
    • 基础选择器又包括：标签选择器、类选择器、id选择器和通配符选择器。

    基础选择器

    标签选择器

    标签选择器（元素选择器）是指用HTML标签名称作为选择器，按标签名称分类，为页面中某一类标签指定统一的CSS样式。

    语法：

    标签名{
    属性1: 属性值1;
    ...
}

    作用：标签选择器可以把某一类标签全部选择出来，比如所有的<div>标签和所有的<span>标签。

    优点：能快速为页面中的同类型标签同一设置样式

    缺点：不能差异化设置样式，只能选择全部的当前标签

    类选择器

    如果根要差异化选择不同的标签，单独选一个或者某几个标签，可以使用类选择器。

    语法：

    <!-- 定义类 -->
.类名{
    属性1: 属性值1;
    ...
}

    使用时要先定义类，再在html中的class属性调用已经定义好的类（样式点定义，结构类调用。一个或多个，开发最常用。）

    注意：

    • 类选择器使用"."来进行标识，后面紧跟类名。
    • 不能使用标签名最为类名。
    • 长名称或词组可以使用"-"来为选择器命名（如：yh-l）
    • 不要使用纯数字、中文命名，尽量使用英文字母来表示。
    • 命名要有意义

    类命名规则：

    类选择器的多类名

    简单理解就是一个标签可以指定多个类名。

    使用方式：多个类名之前用空格分开

    • 可以把一些标签元素相同的样式（共同的部分）放到一个类里面。
    • 这些标签都可以调用这个公共的类，然后再调用自己独有的类。
    id选择器

    id选择器可以为标有特定id的HTML元素指定特定的样式。

    HTML元素以id属性来设置id选择器，CSS中id选择器以“#”来定义。

    语法：

    #id名{
    属性1: 属性值1;
    ...
}

    与类选择器的区别：只能被调用一次。类选择器一般在修改样式时使用，id选择器一般用于页面的唯一性元素上，经常和javascript搭配使用。

    通配符选择器

    在CSS中，通配符选择器使用*定义，它表示选取页面中所有元素（标签）。

    语法：

    * {
    属性1: 属性值1;
    ...
}

    总结

    文本属性

    CSS Text（文本）属性可定义文本的外观，比如文本的颜色、对济文本、装饰文本、文本缩进、行间距等。

    文本颜色

    即color属性。

    颜色属性值的表示方式：

    1. 预定义的颜色值。如pink
    2. 十六进制。如#ff0000，#ff6600
    3. RGB代码。如rgb(255,0,0)，rgb(100%,%0,%0)

    最常用16进制

    文本对齐

    即text-align属性（只能设置水平对齐方式）

    属性值：

    1. left - 左对齐 - 默认
    2. right - 右对齐
    3. center - 居中对齐

    如：

    <style>
    h1 {
        /* 本质是让h1盒子里面的文字水平居中对齐 */
        text-align: center;
    }
</style>

    h1标题的范围实际是以页面的宽为宽，以h1标签规定的的高为高的一个盒子的范围

    装饰文本

    即text-decoration属性。（可以给文本添加下划线、删除线、上划线等）

    属性值：

    1. none - 默认 - 无装饰
    2. underline - 下划线
    3. overline - 上划线
    4. line-through - 删除线

    如：

    <!-- 取消a标签（链接标签默认的下划线） -->
<style>
    a {
        text-decoration: none;
        color: black;
    }
</style>

    文本首行缩进

    即text-indent属性（用于指定文本的第一行缩进，通常是将段落的首行缩进。）

    属性值：

    1. 需要缩进的像素值（可以取负值，即向左缩进）
    2. 需要缩进的相对单位，以em为单位，表示当前元素（font-size）一个文字的大小。

    如：text-indent: 10px;或text-indent: 2em;（表示缩进当前字号两个字的距离）

    行间距（行高）

    即line-height属性

    属性值是像素值。

    行间距的组成：

    行间距包含上间距，文本高度，下间距

    • 上下间距随行间距的变化而变化
    • 文本高度是确定的，不随行间距的变化而变化
    • 上下间距永远是相同的

    CSS的三种样式表

    按照CSS样式书写的位置（或引入的位置），CSS样式表可以分为三大类：

    1. 行内样式表（行内式）
    2. 内部样式表（嵌入式）
    3. 外部样式表（链接式）

    内部样式表

    内部样式表（内嵌样式表）是写到html页面内部.是将所有的CSS代码抽取出来，单独放到一个<style>标签中。

    • <style>标签理论上可以放在HTML文档的任何地方，但一般会放在文档的<head>标签中
    • 通过此种方式，可以方便控制当前整个页面中的元素样式设置
    • 代码结构清晰，但是并没有实现结构与样式完全分离

    在学习与练习中常用

    行内样式表

    行内样式表（内联样式表）是在元素标签内部的style属性中设定CSS样式。适合于修改简单样式。

    即直接在标签的内部写，如：

    <body>
    <div style="color: red; font-size: 12px;">青春不常在，抓紧谈恋爱</div>
</body>

    其中：

    • style其实就是标签的属性
    • 要写在双引号中间，写法要符合CSS规范
    • 只可以控制其所在的标签的样式
    • 由于书写繁琐，并且没有体现出结构与样式相分离的思想，所以不推荐大量使用，只有对当前元素添加简单样式的时候，可以考虑使用

    外部样式表

    实际开发中最常用，也最推荐同学们使用。

    实际开发都是外部样式表，适合于样式比较多的情况。核心是，样式单独写到CSS文件中，之后把CSS文件引入到HTML页面中使用。

    使用方式：

    1. 新建一个后缀名为.css的样式文件，把所有CSS代码都放入此文件中。
    2. 直接在.css文件中写样式即可，不需要写<style>标签
    3. 在HTML页面中，使用<link>标签引入这个文件。即：<link rel="stylesheet" href=".css文件路径">

    3. 如果有父子级关系的标签，可以用＞比如ul>li就可以了
    4. 如果有兄弟关系的标签，用+就可以了比如div+p
    5. 如果生成带有类名或者id名字的，直接写标签名.类名或者标签名#id名再按tab键就可以了
    6. 如果生成的div类名是有顺序的，可以用自增符号＄，如：div.demo%*5
    7. 如果想要在生成的标签内部写内容可以用{}表示，如：div{我叫yhl}

    快速生成CSS样式

    例：

    • 输入tac生成text-align: center;
    • 输入w100生成width: 100px;
    • 输入lh26px生成line-height: 26px;

    元素显示模式

    什么是元素显示模式

    元素显示模式就是元素（标签）以什么方式进行显示，比如<div>自己占一行，比如一行可以放多个<span>。

    作用：网页的标签非常多，在不同地方会用到不同类型的标签，了解他们的特点可以更好的布局我们的网页。

    HTML元素一般分为块元素和行内元素两种类型。

    块元素

    常见的块元素：

    <h1>~<h6>，<p>，<div>，<ul>，<ol>，<li>等，其中<div>是最典型的块元素

    特点：

    • 自己独占一行
    • 可以自行设置高度，宽度，外边距和内边距
    • 宽度默认是容器的（父级宽度）的100%
    • 是一个容器或盒子，里面可以放行内或者块级元素

    注意：

    • 文字类元素内不能使用块级元素。如：<p>标签或<h1>~<h6>标签主要用于存放文字，因此<p>里面不能放块级元素，特别是不能放<div>

    行内元素

    常见的行内元素：

    常见的行内元素有<a>、<strong>、<b>、<em>、<i>、<del>、<s>、<ins>、<u>、<span>等，其中<span>标签是最典型的行内元素。有的地方也将行内元素称为内联元素

    特点：

    • 相邻行内元素在一行上，一行可以显示多个。
    • 高、宽直接设置是无效的。
    • 默认宽度就是它本身内容的宽度。（小盒子）
    • 行内元素只能容纳文本或其他行内元素。行内元素不可以放块元素。

    注意：

    • 链接<a>里面不能再放链接<a>了
    • 有时链接<a>里可以放块级元素，但是给<a>转换一下块级模式最安全

    行内块元素

    常见的行内块元素：

    在行内元素中有几个特殊的标签<img/>、<input/>、<td>，它们同时具有块元素和行内元素的特点，有些资料称它们为行内块元素。

    特点：

    • 和相邻行内块元素在一行上，但是他们之间会有空白缝隙。一行可以显示多个。（行内元素特点）
    • 默认宽度就是它本身内容的宽度。（行内元素特点）
    • 高度，行高、外边距以及内边距都可以控制。（块级元素特点）

    元素显示模式的转换

    特殊情况下，我们需要元素模式的转换，简单理解就是一个模式的元素需要另外一种模式的特性。如，增加链接的点击范围。

    转换为块元素：display: block;

    使用最多

    如：

    <style>
    a {
        width: 150px;
        height: 50px;
        background-color: pink;
        /* 把行内元素 <a> 转换为 块级元素 */
        display: block;
    }
</style>
<body>
    <a href="#">123</a>
</body>
    转换为行内元素：display: inline;

    不常用

    <style>
    div {
        /* 把块元素转换为行内元素 */
        display: inline;
    }
</style>
    转换为行内块元素：display: inline-block;

    可以巧妙使用

    总结

    背景

    通过CSS背景属性，可以给页面元素添加背景样式。

    背景属性可以设置背景颜色、背景图片、背景平铺、背景图片位置、背景图像固定等。

    背景颜色

    即：background-color: color;

    属性值：

    • 默认为transparent（透明）

    背景图片

    background-image属性描述了元素的背景图像。实际开发常见于logo或者一些装饰性的小图片或者是超大的背景图片，优点是非常便于控制位置（精灵图也是一种运用场景）

    即：background-image

    属性值：

    • none - 无背景图 - 默认
    • url - 使用绝对或相对地址指定背景图像

    背景平铺

    即：background-repeat: 属性值;

    属性值：

    • repeat - 平铺 - 默认
    • no-repeat - 不平铺
    • repeat-x - 延x轴平铺
    • repeat-y - 延y轴平铺

    如：

    <style>
    div {
        /* 不平铺 */
        background-repeat: no-repeat;
        background-color: pink;
        /* 页面元素既可以添加背景颜色也可以添加背景图片只不过背景图片会压住背景颠色 */
    }
</style>

    背景图片的位置

    即：background-position属性

    属性值：x，y

    如：background-position: x y;

    其中：

    参数代表的意思是：x坐标和y坐标。可以使用方位名词或者精确单位

    1. 参数是方位名词。如right，left，center，top等
    • 如果指定的两个值都是方位名词，则两个值前后顺字无关，比如left top和top left效果一致。如：background-position: center top;和background-position: top center;等价。
    • 如果只指定了一个方位名词，另一个值省略，则第二个值默认居中对齐。如果第一个值是top（y轴值）则第二个参数省略x（水平居中对齐）
    2. 参数是精确单位
    • 如果参数值是精确坐标，那么第一个肯定是x坐标，第二个一定是y坐标。
    • 这里坐标表示的是图片左侧到盒子左侧的距离和图片顶部到盒子顶部的距离
    • 如果只指定一个数值，那亥数值一定是×坐标，另一个默认垂直居中

    如：background-position: 50px 20px;

    3. 参数是混合单位
    • 如果指定的两个值是精确单位和方位名词混合使用，则第一个值是×坐标，第二个值是y坐标

    背景图像固定

    即：background-attachment。设置背景图像是否固定或者随着页面的其余部分滚动。

    background-attachment后期可以制作视差滚动的效果。

    属性值：

    • scroll - 背景图像是随对象内容滚动 - 默认
    • fixed - 背景图像固定

    背景属性的复合写法

    为了简化背景属性的代码，我们可以将这些属性合并简写在同一个属性background中。从而节约代码量。

    当使用简写属性时，没有特定的书写顺序，一般习惯约定顺序为：

    background：背景颜色、背景图片地址、背景平铺、背景图像滚动、背景图片位置；

    如：

    <style>
    body {
        background: black url(image.jpg) no-repeat fixed center top;
    }
</style>

    在实际开发中我们更提倡简写

    背景颜色半透明

    即：background: rgba(0, 0, 0, 0.3);

    其中：

    • 最后一个数值表示透明度。取值在0～1之间。0表示100%透明，1表示100%不透明。
    • 我们习惯把0.3的0省略掉，写为background:rgba(0,0,0.3);
    • 注意：背景半透明是指盒子背景半透明，盒子里面的內容不受景响
    • 半透明是CSS3新增属性，是IE9+版本浏览器才支持的（现在已经不关注兼容性了，放心大胆使用就好）

    CSS三大特性

    CSS盒子模型

    页面布局要学习三大核心，盒子模型，浮动和定位.学习好盒子模型能非常好的帮助我们布局页面。

    看透网页布局的本质

    每个网页都是由许许多多的盒子组成的

    网页布局过程：

    1. 先准备好相关的网页元素，网页元素基本都是盒子Box。
    2. 利用CSS设置好盒子样式，然后摆放到相应位置。（核心）
    3. 往盒子里面装内容。

    盒子模型的组成

    所谓盒子模型：就是把HTML页面中的布局元素看作是一个矩形的盒子，也就是一个盛装內容的容器。

    CSS盒子模型本质上是一个盒子，封装周围的HTML元素，它包括：边框、外边距、内边距、和实际内容

    边框（border）

    border可以设置元素的边框。边框有三部分组成；边框宽度（粗细）、边框样式、边框颜色。

    属性：

    • border-width - 边框粗细 - 单位是px
    • border-style - 边框样式
    • border-color - 边框颜色

    属性值：

    1. border-width：...px
    2. border-style：solid（实线边框）、dashed（虚线边框）、dotted（点线边框）
    3. border-color：color

    语法：

    border: border-width border-style border-color

    注意：边框的复合写法没有顺序！以上顺序为通俗约定。

    边框的分开写法：

    即把border换为border-top/bottom/left/right

    边框会额外增加盒子的实际大小：

    边框会额外增加盒子的实际大小。因此我们有两种方案解决：

    1. 测量盒子大小的时候，不量边框.
    2. 如果测量的时候包含了边框，则需要width/height减去边框宽度

    内容

    文字，图片，其他的盒子

    内边距（padding）

    属性：

    • padding-left - 左内边距
    • padding-right - 右内边距
    • padding-top - 上内边距
    • padding-bottom - 下内边距

    属性值为像素值。

    内边距的复合写法：

    padding会额外增加盒子的实际大小：

    如果保证盒子跟效果图大小保持一致，则让width/height减去多出来的内边距大小即可。

    如果盒子本身没有指定width/height属性，则此时padding不会撑开盒子大小。

    外边距（margin）

    margin属性用于设置外边距，即控制盒子和盒子之间的距离。

    属性：

    margin的简写方式与padding完全一致。

    外边距的典型应用：让块级盒子水平居中

    条件：（步骤）

    1. 盒子必须指定了宽度。
    2. 保证盒子左右的外边距都设置为auto。

    常见写法：

    • margin-left auto; margin-right auto;
    • margin: auto;
    • margin: 0 auto;

    注意：以上方法是让块级元素水平居中，行内元素或者行内块元素水平居中给其父元素添加 text-align:center即可。（把行内元素或行内块元素当成普通的文字来看）

    外边距的合并
    嵌套块元素垂直外边距的塌陷

    对于两个嵌套关系（父子关系）的块元素，父元素有上外边距同时子元素也有上外边距，此时父元素会塌陷较大的外边距值。

    解决方案：

    • 可以为父元素定义上（透明）边框。
    • 可以为父元素定义上内边距。
    • 可以为父元素添加overflow:hidden。（先记住，不管原理，最好用的方案）

    清楚网页元素自带的边距

    网页元素很多都带有默认的内外边距，而且不同浏览器默认的也不一致。因此我们在布局前，首先要清除下网页元素的内外边距。

    <style>
    * {
        /* 这句话也是我们CSS的第一行代码 */
        padding:0;
        margin:0;
    }
</style>

    注意：行内元素为了照顾兼容性，尽量只设置左右内外边距，不要设置上下内外边距。但是转换为块级和行内块元素就可以设置上下内外边距了。

    圆角边框

    语法：

    border-radius: length;

    参数值：

    • 像素值。
    • 百分比。如：50%的意思就是半径为宽度和高度的一半。等价于将矩形完全变为（椭）圆。

    参数值的复合写法：属性后可以跟四个值，分别代表左上角、右上角、右下角、左下角。

    也可以分开写：分开写：border-top-left-radius、border-top-right-radius、border-bottom-right-radius和border-bottom-left-radius

    其中：

    • radius是半径的意思
    • 原理：（椭）圆与边框的交集形成圆角效果
    • 如果是正方形，想要设置为一个圆，把数值修改为高度或者宽度的一半即可，或者直接写为50%
    • 如果是个矩形，设置为高度的一半即可

    阴影

    盒子阴影

    语法：

    box-shadow: h-shadow v-shadow blur spread color inset;

    属性值：

    单位为像素和颜色

    其中颜色属性的属性值一般设置为rgba(0, 0, 0, .3)（rgb中(0，0，0)是黑色）

    原先盒子没有影子，当鼠标经过时出现影子（未完待续）

    文字阴影

    语法：

    text-shadow: h-shadow v-shadow blur color;

    属性值：

    浮动

    • 普通流（标准流）
